eOS Specification

This section provides technical details about the eOS proof-of-concept setup for the EU public sector.

Base System

  • Fedora 42 Linux as the core operating system.
  • GNOME as the default desktop environment.
  • Atomic layered architecture for modular deployments.

Container Technologies

  • Podman installed by default for container management.
  • Docker optional, manual installation required if needed.
  • Supports container orchestration and sandboxing for secure workloads.

Hardware Requirements

  • Minimum: 2 GB RAM, 1 GHz CPU, 15 GB storage
  • Recommended: 4 GB RAM, modern dual-core CPU, 25 GB+ storage
  • Display: 1024×768 minimum, 1080p recommended

Software Stack

  • Default applications: Firefox, LibreOffice, GNOME Terminal, Podman
  • Package management: DNF, Flatpak
  • Security: SELinux enforced, no telemetry

Deployment Notes

  • Supports installation on typical public sector PCs and laptops.
  • ISO image available for live testing before installation.
  • Modular architecture allows adding national, regional, or organizational layers.
